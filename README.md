# ATM Interface

## Objective

The objective of this project is to build a simple ATM Interface using Java.

## Features

* Login using User ID and PIN
* Transaction History
* Withdraw Money
* Deposit Money
* Transfer Money
* Check Balance
* Quit Option

## Tools Used

* Java
* VS Code
* JDK

## Steps Performed

1. Created login page.
2. Added ATM menu.
3. Implemented withdraw and deposit.
4. Added transfer functionality.
5. Displayed transaction history.

## Outcome

The project was successfully created and tested in Java.
